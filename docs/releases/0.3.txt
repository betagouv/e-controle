## Instructions de mise à jour

1. Configurer les "locales" comme ceci:

    localedef -c -f UTF-8 -i fr_FR fr_FR.UTF-8


1. Changer les locales dans le fichier `.env`

    export LANG=fr_FR.UTF-8
    export LC_ALL=fr_FR.UTF-8
