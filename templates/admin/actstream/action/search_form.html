Pour trouver toutes les actions liées à :
  • l'utilisateur dont l'ID est 123: /actstream/action/?actor_object_id=123
  • le controle dont l'ID est 456: /actstream/action/?target_object_id=456
  • les deux : /actstream/action/?actor_object_id=123&target_object_id=456
Comment trouver l'ID d'un objet ?
Aller à la page de l'objet et regarder le numéro dans l'URL.
Exemple : trouver l'ID du contrôle "CCG de Blablabla". Je vais sur la page du CCG de Blablabla dans l'admin. L'URL est : /control/control/37/change/. L'ID est donc 37.
